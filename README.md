## Features

### Calculator

| Feature              | Description                                                              |
|----------------------|--------------------------------------------------------------------------|
| **Basic Calculator** | Addition, subtraction, multiplication, division with auto-sizing display |
| **Percentage**       | Quick percentage calculations                                            |
| **History**          | Review and reuse past calculations                                       |
| **Adaptive UI**      | Optimized layout for phones, tablets & landscape                         |

### Investment Calculators

| Screen                    | Description                                                                                          |
|---------------------------|------------------------------------------------------------------------------------------------------|
| **SIP Calculator**        | Plan systematic investments with step-up & inflation adjustment. Shows invested vs returns pie chart |
| **SWP Calculator**        | Calculate systematic withdrawal amounts from your corpus                                             |
| **FD Calculator**         | Fixed deposit maturity amount based on principal, rate & tenure                                      |
| **Lump Sum Calculator**   | Future value of a one-time investment with visual breakdown                                          |
| **CAGR Calculator**       | Compound Annual Growth Rate between initial and final values                                         |
| **Dividend Calculator**   | Analyze stock dividend income from your investments                                                  |
| **Retirement Calculator** | Estimate the corpus needed for a comfortable retirement                                              |
| **Inflation Calculator**  | See how inflation erodes purchasing power over time                                                  |

### Loan Calculators

| Screen                 | Description                                                 |
|------------------------|-------------------------------------------------------------|
| **EMI Calculator**     | Monthly loan repayment amount with interest breakdown       |
| **Loan Affordability** | Maximum loan you can qualify for based on income & expenses |
| **Loan Prepayment**    | Interest saved by making extra payments towards your loan   |

### Tax & Shopping

| Screen                   | Description                                   |
|--------------------------|-----------------------------------------------|
| **Tax / GST Calculator** | Add or remove GST and estimate tax amounts    |
| **Discount Calculator**  | Calculate final price after discount savings  |
| **Bill Splitter**        | Split bills and tips evenly across your group |

### Health Calculators

| Screen                      | Description                                                                      |
|-----------------------------|----------------------------------------------------------------------------------|
| **BMI Calculator**          | Body Mass Index with weight category interpretation                              |
| **BMR Calculator**          | Basal Metabolic Rate — daily calorie needs based on height, weight, age & gender |
| **Water Intake Calculator** | Daily hydration needs based on weight & activity level                           |

### Utilities

| Screen                    | Description                                                        |
|---------------------------|--------------------------------------------------------------------|
| **Unit Converter**        | Convert between length, weight & area units                        |
| **Age & Date Calculator** | Exact age in years/months/days & date difference between two dates |
